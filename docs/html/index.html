Convolutional LSTM module.

forward(input_tensor, hidden_state=None)

Perform a forward pass of the ConvLSTM module.

Parameters:

  • input_tensor (torch.Tensor) – Input tensor of shape: (b, t, c, h, w) or (t, b, c, h, w).

  • hidden_state (list or None) – List of initial hidden and cell states for each layer.

Returns:

Tuple containing two lists of length num_layers (or length 1 if return_all_layers is False):

  1. layer_output_list: List of output tensors for each layer.

  2. last_state_list: List of last hidden states and cell states

    for each layer.

Return type:

tuple
