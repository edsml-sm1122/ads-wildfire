KalmanGain¶

wildfire.KalmanGain(B, H, R)¶

Calculates the Kalman gain matrix.

Args:
B (numpy.ndarray): Background error covariance matrix with

shape (n, n).

H (numpy.ndarray): Measurement matrix with shape (p, n). R (numpy.ndarray): Measurement error covariance matrix

with shape (p, p).

Returns:

numpy.ndarray: Kalman gain matrix with shape (n, p).
